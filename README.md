# Book-Website-Scraper
Python script that scrapes data on all of the book, sorted by category, from books.toscrape.com.

# HOW TO USE
1. Create a Virtual Environment to store the dependencies. Open VSCode and type "python -m venv .venv" into a terminal. There will be a pop-up asking if you would like to select it as your new workspace, click **Yes**.
2. Open a new terminal and run "pip install -r requirements.txt"
3. Run allcategories.py and the script will create csv_files and img_files folders that will store the scraped data.
\nIt's not recommended to run either singlecategory or singleproduct as they do not put the files in the correct folders and it can make a lot of unorganized files if you aren't careful.
